Fruit Segmentation and Classification using GLCM Texture Analysis and DBSCAN
Overview

This project presents a texture-driven approach for fruit segmentation and classification using:

Gray-Level Co-occurrence Matrix (GLCM)
Autocorrelation and Homogeneity Features
Custom DBSCAN Clustering Algorithm
Morphological Opening and Closing Operations

Unlike conventional image classification methods that rely heavily on color information, this project focuses on texture analysis to distinguish fruit classes and perform segmentation in both controlled and real-world environments. The methodology was developed as part of a Computer Vision assignment at Deggendorf Institute of Technology.
